# Search Engine Tool

A Python-based search engine tool that crawls [quotes.toscrape.com](https://quotes.toscrape.com/), builds an inverted index with TF-IDF scoring, and provides an interactive CLI for searching.

## Project Overview

Developed for COMP3011 (Web Services and Web Data) Coursework 2 at University of Leeds.

- **Web Crawling**: Crawls all quote pages and author detail pages with a 6-second politeness window. Failed URLs are collected and retried in a final pass.
- **Inverted Indexing**: Builds an inverted index storing word statistics (frequency, positions, TF) for each document. Computes TF-IDF scores for relevance ranking.
- **Search**: AND-logic multi-word queries ranked by TF-IDF. Auto-suggests similar terms when no results are found.

## Architecture

```
Website --> Crawler (fetch & parse) --> Raw crawl data
                                          |
                                    Indexer (tokenize, build index, compute TF-IDF)
                                          |
                                    SearchEngine (query, rank, suggest)
                                          |
                                    CLI (interactive shell)
```

**Document granularity**: Each page URL is one document (all quotes on that page combined). Each author page is a separate document (name, birth info, and biography indexed).

## Usage

```bash
python -m src.main
```

### Commands

#### `build` - Crawl website and build index
```
> build
Crawling https://quotes.toscrape.com/ ...
  Page 1: 10 quotes found
  Page 2: 10 quotes found
  ...
Building index...
  Indexed 312 unique terms across 60 documents
Saving index to data/index.json ...
Done!
```

#### `load` - Load a previously built index
```
> load
Loading index from data/index.json ...
Index loaded: 312 terms, 60 documents
Ready for queries.
```

#### `print <word>` - Show inverted index entry
```
> print world
Index entry for "world":
  Found in 3 document(s) (IDF: 1.234)

  1. https://quotes.toscrape.com/page/1/
     Frequency: 2, Positions: [3, 15]
     TF: 0.0312, TF-IDF: 0.0385
```

#### `find <word1> [word2 ...]` - Search with AND logic
```
> find good friends
Searching for: good AND friends
Found 2 result(s):

  1. [Score: 0.0842] https://quotes.toscrape.com/page/3/
  2. [Score: 0.0531] https://quotes.toscrape.com/page/7/
```

When a term is not found, suggestions are shown automatically:
```
> find physic
No results found: term 'physic' does not exist in the index.
  Did you mean: physicist, physics, physical?
```

```bash
python -m pytest -v

python -m pytest --cov=src --cov-report=term-missing
```

## Project Structure

```
search-engine-tool/
  src/
    models.py      # Data models (WordStats, DocumentMeta, SearchResult)
    crawler.py     # Web crawler with politeness control and retry queue
    indexer.py     # Inverted index builder with TF-IDF scoring
    search.py      # Search engine with AND logic and suggestions
    main.py        # Interactive CLI entry point
  tests/
    conftest.py    # Shared fixtures and sample HTML
    test_crawler.py
    test_indexer.py
    test_search.py
    test_main.py
  data/
    index.json     # Compiled index file (generated by build)
  requirements.txt
  README.md
```

## Dependencies

- `beautifulsoup4` - HTML parsing
- `pytest` - Testing framework
- `pytest-cov` - Coverage reporting
